SASS-setup:
Consern:
Which version of Tailwind compatable with SASS


Short answer: Yes — Tailwind works fine with SASS. 
Recommended pipeline: 
    compile SASS first, 
    then run PostCSS/Tailwind on the compiled CSS 
    (SASS → PostCSS/Tailwind → final CSS).

Why this order

    Tailwind (PostCSS) needs to see plain CSS with Tailwind directives 
    and any @apply rules so it can expand utilities. 
    SASS compilers don't understand Tailwind directives, 
    so compile SASS first, then let Tailwind/PostCSS process the output.
